Fuel 是一个基于 SFC（Desert Knowledge Manual）和加载的扩展（mods）的 PPTX 合并器。它通过加载一个 jar 文件并将内部 PPTX 文件插入 fuel.json 文件的第二行（语法完全非正式）来实现合并。它基于 Python 并需要 pptx 库。
